Instalarea programelor necesare
Este necesar sa instalați următoarele programe pentru a rula ProManager pe calculatorul dumneavoastră
1.	Visual Studio Code -Folosit pentru a putea scrie și edita cod, accesați site-ul oficial Visual Studio Code https://code.visualstudio.com/ de unde puteți descărca editorul pentru sistemul de operare pe care îl utilizați.
2.	XAMPP -Puteți descarcă ultima versiune de pe siteu-l oficial https://www.apachefriends.org/  pentru a rula serverul apache si a avea acces la mysql.
3.	.Codul sursa -Pentru a accesa codul sursă al proiectului ProManager vizitați linkul ---sau accesați secțiunea Repositories de pe profilul meu de GitHub---. Găsiți proiectul sub numele ProManager. Pentru a-l descărca, faceți clic pe butonul "Code" și selectați "Download ZIP". Salvați arhiva în directorul htdocs din folderul xampp, localizat în partiția unde este instalat sistemul de operare. De exemplu, în cazul meu, calea este A:\xampp\htdocs deoarece am instalat XAMPP în partiția A.

După instalarea XAMPP pe noul PC și pornirea serviciilor Apache și MySQL, deschideți phpMyAdmin accesând http://localhost/phpmyadmin. Creați o nouă bază de date cu același nume ca pe PC-ul original(promanager). Selectați noua bază de date, apoi mergeți la tab-ul "Import" și alegeți fișierul SQL exportat, după care faceți clic pe "Go" pentru a importa baza de date. Deschideți fișierul de configurare al proiectului db.php (Figura 10) in cadul Visual Studio Code și verificați ca setările pentru conexiunea la baza de date (host, user, password, database name) să fie corecte pentru noul mediu.
